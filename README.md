## 项目介绍

本项目名称为易知耳（E2Know），为用户提供快捷的耳机参数查询和耳机讨论平台。

### 技术介绍

采用VUE框架,Node.js,mysql,element-ui,echarts等技术

### 开发目的

#### 为用户提供低成本的耳机查询服务

#### 为用户提供自由客观的耳机讨论平台
